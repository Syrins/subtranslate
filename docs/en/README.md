# SubTranslate Documentation (EN)

SubTranslate is a platform for subtitle translation and video export workflows.

This English documentation set explains backend, frontend, engine, and deployment details.

## Project Summary

- Upload video or subtitle files
- Extract subtitle tracks from videos
- Translate subtitles with AI engines
- Edit subtitle lines in a synced editor
- Export in burn-in or soft-sub mode
