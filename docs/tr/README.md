# SubTranslate Dokumantasyon (TR)

SubTranslate, video ve altyazi dosyalarini cevirip export etmeyi kolaylastiran bir platformdur.

Bu dokumantasyon seti; backend, frontend, engine ve deploy konularini Turkce olarak aciklar.

## Proje Ozeti

- Video veya altyazi yukleme
- Videodan altyazi track cikarma
- AI ile ceviri (OpenAI, DeepL, Gemini)
- Editor ekraninda satir bazli duzenleme
- Burn-in veya soft-sub export
