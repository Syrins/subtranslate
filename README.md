# SubTranslate

SubTranslate is a full-stack platform for subtitle translation and video export.

SubTranslate, altyazi cevirisi ve video export icin gelistirilmis full-stack bir platformdur.

## What It Does

- Upload video or subtitle files
- Extract subtitle tracks from videos
- Translate subtitles with AI engines
- Edit subtitle lines in a synced editor
- Export as burn-in or soft-sub output
