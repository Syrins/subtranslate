┌──────────────────────────────────────────────────────────────────────┐
│ QUICK START                                                          │
├──────────────────────────────────────────────────────────────────────┤
│                                                                      │
│ Using Docker (recommended):                                          │
│                                                                      │
│   git clone https://github.com/Syrins/subtranslate.git             │
│   cd subtranslate                                                    │
│   cd backend && cp .env.example .env  # Add your API keys           │
│   cd .. && docker-compose up -d                                     │
│   docker-compose exec backend npm run prisma:migrate                │
│   docker-compose exec backend npx prisma db seed                    │
│                                                                      │
│   Access: http://localhost                                          │
│                                                                      │
│ Without Docker:                                                      │
│                                                                      │
│   npm install                                                        │
│   cd backend && cp .env.example .env  # Configure                   │
│   npm run prisma:migrate && npx prisma db seed                      │
│   cd .. && npm run dev                                              │
│                                                                      │
│   Frontend: http://localhost:5173                                   │
│   Backend:  http://localhost:3000                                   │
│                                                                      │

│ USER WORKFLOW                                                        │
├──────────────────────────────────────────────────────────────────────┤
│                                                                      │
│ 1. Register & Login                                                  │
│    └─> JWT authentication, bcrypt password                          │
│                                                                      │
│ 2. Create Project                                                    │
│    └─> Upload MKV or provide video link                             │
│         └─> Auto-extract subtitles (background job)                 │

│ 3. Translate Subtitles                                               │
│    └─> Choose: OpenAI, DeepL, or Gemini                            │
│         └─> AI-powered translation                                  │

│ 4. Edit & Style                                                      │
│    └─> Live preview editor                                          │
